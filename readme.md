# JavaScript Comeback Roadmap

A structured roadmap to rebuild JavaScript syntax, improve repetition, and gradually transition into usable projects with HTML, CSS, and JavaScript.

## Chapter 1 — Loop Mastery

### Focus
- for loops
- array indexing
- conditions inside loops

### Main Syntax Repetition
```js
for (let i = 0; i < arr.length; i++) {

}
```

### Project
Array Analyzer

### Tasks
- Find the largest number
- Find the smallest number
- Count how many times a number appears
- Return all even numbers
- Return all numbers greater than X

### Rules
- No `.map()`
- No `.filter()`
- No shortcuts
- Use only `for` loops

### Outcome
Build comfort with:
- traversing arrays
- reading values
- conditional logic in loops

## Daily Practice Structure

### Daily Routine
- 20 minutes syntax repetition
- 30 minutes project work
- 10 minutes code cleanup

### Weekly Routine
- 1 chapter per week
- repeat key functions
- rewrite from memory

## Rules for Success

### Do
- Repeat the same syntax often
- Build without copying
- Rewrite old code
- Focus on understanding

### Avoid
- Jumping ahead too fast
- Overdesigning UI
- Watching too many tutorials
- Memorizing without practice

## End Goal

By the end of this roadmap you should:
- rebuild JavaScript confidence
- improve syntax recall
- strengthen logic skills
- build usable projects
- feel ready for harder coding challenges
